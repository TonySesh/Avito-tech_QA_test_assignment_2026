### BUG-001: API принимает отрицательные значения в числовых полях
- **ID:** BUG-001
- **Связанные тест-кейсы:** TC-POST-23, TC-POST-24, TC-POST-25, TC-POST-26, TC-POST-27
- **Описание:** API создаёт объявление с отрицательными значениями в полях sellerID, price, likes, viewCount, contacts
- **Шаги воспроизведения (для price):**
1. Отправить POST-запрос на /api/1/item
2. В теле запроса передать JSON с "price": -100
3. Посмотреть ответ
- **Фактический результат:** Статус 200, объявление создаётся
- **Ожидаемый результат:** Статус 400, объявление не создаётся
- **Серьёзность:** High
- **Окружение:** qa-internship.avito.com
